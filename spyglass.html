Spyglass

Market Intelligence & Options Analysis Platform

Python GitHub Actions Discord

Spyglass is a Python-based screening engine that runs unattended daily analysis across equities, classifies market regimes, and generates systematic trading signals and options strategy alerts. Results are delivered automatically through Discord — no manual intervention required.

How It Works

Market Regime Classification

A five-regime market classifier evaluates current conditions before any signal is generated. Position sizing and strategy selection adapt to the classified regime — the same strategy doesn't run identically in trending and mean-reverting environments.

Options Strategy Engine

Full lifecycle management of the options wheel strategy: identifies candidates for cash-secured puts, tracks through potential assignment, manages covered calls, and handles rolling or expiration decisions. Each stage is governed by risk-based position sizing.

Automated Operations

Unattended daily execution on GitHub Actions, gated by an NYSE trading-calendar check. Three-channel Discord alerting delivers signals, status updates, and P&L summaries. Batch data processing reduced screening runtime from ~10 minutes to under 2.
